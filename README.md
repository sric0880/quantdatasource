# quantdatasource

finance datasource downloader and importer.

无严格测试，请谨慎使用

主要数据源：

- [Tushare](https://tushare.pro/)
- [TQSDK](https://github.com/shinnytech/tqsdk-python)

导入数据库：

- MongoDB
- TDengine
- DuckDB

## Usage

```sh
python -m quantdatasource.jobs -- --help
```

or

```sh
python -m quantdatasource -- --help
```
